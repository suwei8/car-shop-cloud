# 车店云管家 v0.1.0

## 功能概览

### 管理后台 (Web)
- 工作台数据面板
- 客户档案管理
- 车辆档案管理（车型库联动选择）
- 工单管理（接车开单、状态流转）
- 派工管理
- 配件管理（供货商、质保）
- 库存管理
- 结算收款
- 储值卡管理
- 营业日报、技师产值
- 员工管理、角色权限
- 平台后台（商户管理、套餐管理）

### 安卓 APP
- 员工登录
- 我的派工
- 工单详情
- 储值卡查询

## 技术栈

- 后端：NestJS + Prisma + PostgreSQL
- 前端：Vue 3 + Element Plus
- 移动端：uni-app (H5 + Android WebView)
- 部署：Docker Compose
