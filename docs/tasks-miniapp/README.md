## 通用要求（每个任务都适用）

1. 先读根目录 `AGENTS.md`，遵守硬约束：租户隔离、金额 Decimal、资金/库存写流水、已结算工单不可直接改。
2. 严格按任务书范围执行，不做范围外「顺手优化」。
3. 不引入车主端/车主小程序相关内容（已下线）。
4. 涉及后端确保 `pnpm build:api` 通过；涉及 web 确保 `vue-tsc` 通过；涉及小程序确保 `build:mp-weixin` 可编译；新增/调整测试通过。
5. 完成后把回执填入各任务书「回执区域」，不得改动任务书其他章节。
6. Prisma 迁移只在本机/测试库执行，严禁操作生产库。
